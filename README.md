# tmpdir

Provide the newest `os.tmpdir` implementation and work in every version of node.

[![travis](https://travis-ci.org/dominictarr/tmpdir.png?branch=master)

## Installation

```
npm install tmpdir
```

## Usage

```
var tmpdir = require('tmpdir')()
```
